# Intune Endpoint Lifecycle & Hardware Governance

## Overview
This repository demonstrates an enterprise style Microsoft Intune implementation focused on endpoint management, security compliance, automation, and hardware lifecycle governance.

The goal of this project is to reflect how large organizations manage and secure endpoint fleets at scale while maintaining visibility into hardware lifecycle and refresh planning.

---

## Key Focus Areas
- Microsoft Intune endpoint administration
- Windows Autopilot onboarding
- Device compliance and Conditional Access
- PowerShell automation and remediation
- Hardware lifecycle tracking and governance

---

## Why This Project
This project aligns with an Endpoint Hardware Lifecycle Engineer role and demonstrates hands-on experience with Intune configuration, endpoint security, automation, and lifecycle ownership.
